Model Neutral Model of Stone Raw Material Procurmenet (ver. 1.0)

This is a first implementation of

Brantingham, P. Jeffrey (2003): A Neutral Model of Stone Raw Material Procurement. In American Antiquity 68 (3), pp. 487–509. DOI: 10.2307/3557105.

by
Frederik Schaff (frederik.schaff@rub.de)

The statistics are yet incomplete and the description is yet insufficient.

The model makes use of the PajekFromCpp library (here a fixed version) and allows
to create a dynamical network that can be converted to a dynamic svg animation.

See https://github.com/FrederikSchaff/PajekFromCpp for details
