LSD Data Browser

The LSD Data Browser allows exploring the model by inspecting each individual instance of any Object in the LSD model structure and also editing the associated initial data. This is different from the perspectives offered by the main LSD Browser and the LSD Model Structure, which present the model structure by means of Object types. Therefore, these windows do not distinguish among individual instances of each Object, as the Data Browser does. This is an important tools because in many situations the user may be interested in observing each individual Object instance and its content.

The LSD Data Browser window is derived from the LSD Debugger window but does not have the options to deal with simulation runs. Please refer to the manual pages on the LSD Debugger for the functionalities of the LSD Data Browser.
