Model Test_0001_tush (ver. 1.0)

# Description
This testmodel is a meta-test model. Its purpose is to test the utilities
that are used within the tush tests.

# Test procedure

Firstly, we make the specific no window version.
$ rm -f ../lsdNW -f >> tushLogNull.log
$ cd .. && make -f makefileNWtush >> tushLogNull.log

Secondly, we run the tushTest.lsd and expect it to not fail. Important: Pipe stdout to null, to ignore it.
Otherwise you would, e.g., include timing issues.
$ cd .. && ./lsdNW -f tushTest.lsd >> tushLogNull.log

Thirdly, we compare the output with the expected output. Here we use tush explicitly.
